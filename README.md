## emotionWeb

◆日本語テキスト感情分析：Webアプリ化しました

Pythonで書いてます。

こんにちは。
これは、独自の方法で日本語の印象・感情を分析し、表示するソフトです。
何かの参考になれば幸せます。

単語ごとに算出し、行全体の感情を判定します。


コーパスについて 
聖書66巻（日本語及び英語）をベースに、十数万行の感情タグ付きコーパスとしました。
感情は、プラッチクの「感情の環」から１６種類を選択しました。
